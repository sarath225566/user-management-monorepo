# User Management System (Monorepo)

## Tech Stack:
- **Frontend:** Angular 17+ (Bootstrap for UI)
- **Backend:** Spring Boot (Spring Data MongoDB, Web)
- **Database:** MongoDB
- **Architecture:** Monorepo (Backend & Frontend in one repository)

## Features:
- User Registration with validation.
- Real-time data updates and deletions.
- Toast notifications for success/error messages.
- Responsive UI design.
